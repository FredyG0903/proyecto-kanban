# Carpeta de Archivos Estáticos

Esta carpeta contiene archivos estáticos que se sirven desde la raíz de la aplicación.

## Imagen de Fondo

Coloca tu imagen de fondo aquí con el nombre que desees, por ejemplo:
- `background.jpg`
- `fondo.png`
- `background-image.webp`

Luego, en los archivos de componentes, referencia la imagen usando:
```javascript
backgroundImage: 'url(/background.jpg)'
```

**Nota:** La ruta debe comenzar con `/` para referenciar archivos en la carpeta `public`.
